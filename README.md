# EE-352-AUTOMATIC-CONTROLS

Detailed overview on automatic controls theory and applications uing python and MATLAB.

## Contents
1. Introduction to Automatic Controls
2. Mathematical Modeling of Dynamic Systems
3. Transfer Functions and Block Diagrams
4. Time-Domain Analysis of Control Systems
5. Stability Analysis
6. Frequency-Domain Analysis
7. Design of Control Systems
8. State-Space Analysis and Design
9. Digital Control Systems
10. Advanced Control Strategies
11. Applications of Automatic Control Systems

## Getting Started
To get started with the code and examples provided in this repository, follow these steps:
1. Clone the repository to your local machine using:
   ```
   git clone https://github.com/yourusername/EE352-Automatic-Control.git
    ```
2. Navigate to the project directory:
    ```
    cd EE352-Automatic-Control
    ```
3. Install any required dependencies. For Python, you can use:
    ```
    pip install -r requirements.txt
    ```
4. Open the Jupyter notebooks or MATLAB files to explore the examples and exercises.
5. Follow along with the lectures and complete the exercises provided in each section.
