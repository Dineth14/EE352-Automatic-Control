Fundamental Equations

Closed-Loop Transfer Function
$$T(s) = \frac{G(s)}{1 + G(s)H(s)}$$
Physical Meaning

The output-to-input ratio of a negative feedback system. Feedback reduces sensitivity and improves stability.

Mason's Gain Formula
$$T = \frac{1}{\Delta}\sum_{k} P_k \Delta_k$$
Physical Meaning

Computes the transfer function directly from a signal flow graph by enumerating all forward paths and loop gains.

Root Locus Angle Condition
$$\angle G(s)H(s) = \pm 180°(2k+1)$$
Physical Meaning

A point s lies on the root locus if and only if the total angle contribution from all poles and zeros sums to an odd multiple of 180°.

Nyquist Stability Criterion
$$N = Z - P$$
Physical Meaning

The number of clockwise encirclements of −1 equals the difference between closed-loop unstable poles and open-loop unstable poles.

Algebraic Riccati Equation
$$A^TP + PA - PBR^{-1}B^TP + Q = 0$$
Physical Meaning

The matrix equation whose solution yields the optimal LQR gain. Balances state regulation cost Q against control effort cost R.

Kalman Gain
$$K_f = P^{-}C^T(CP^{-}C^T + R_v)^{-1}$$
Physical Meaning

The optimal weighting that fuses noisy measurements with the model prediction, minimizing mean-squared estimation error.
