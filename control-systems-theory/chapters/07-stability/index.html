System Stability

Routh-Hurwitz & Lyapunov Methods

1. Routh-Hurwitz Criterion

A continuous-time Linear Time-Invariant (LTI) system is absolutely stable if and only if all roots of its characteristic polynomial have strictly negative real parts (all poles in the Left-Half Plane).

The Routh-Hurwitz criterion allows us to determine the number of Right-Half Plane (RHP) roots without explicitly factoring the polynomial.

Characteristic polynomial:

$$ P(s) = a_n s^n + a_{n-1} s^{n-1} + \dots + a_1 s + a_0 = 0 $$ (7.1)

We construct the Routh Array. The number of sign changes in the first column equals the number of roots in the RHP.

2. Stability Regions (BIBO)

Bounded-Input Bounded-Output (BIBO) stability requires that for any bounded input $u(t) \le M < \infty$, the output $y(t)$ must also be bounded. This directly equates to the impulse response being absolutely integrable:

$$ \int_0^\infty |h(t)| dt < \infty \quad \iff \quad \text{All poles in LHP} $$ (7.2)
