Chapter 02

Block Diagram Reduction

Signal flow, feedback loops, and Mason's Gain Formula

Chapter Overview

Block diagrams are the visual language of control systems. This chapter teaches you to reduce complex interconnections of transfer functions into a single equivalent transfer function using algebraic rules and Mason's Gain Formula.

Learning Objectives

  • Draw and interpret block diagrams with summing junctions and pickoff points
  • Apply series, parallel, and feedback reduction rules
  • Move pickoff points and summing junctions across blocks
  • Construct signal flow graphs from block diagrams
  • Apply Mason's Gain Formula to determine overall transfer functions

◈ 2.1 — Block Diagram Components

A block diagram is a graphical representation of a system showing the functional relationships between components. The fundamental elements are:

  • Blocks — represent transfer functions $G(s)$. Output = $G(s) \times$ Input.
  • Summing junctions (circles with ±) — add or subtract signals.
  • Pickoff points (branch points) — where a signal is tapped without modification.
  • Arrows — indicate signal flow direction.

◈ 2.2 — Block Diagram Algebra

Series (Cascade) Connection

Series Rule $$G_{eq}(s) = G_1(s) \cdot G_2(s) \cdots G_n(s)$$

Blocks in series: multiply their transfer functions.

Parallel Connection

Parallel Rule $$G_{eq}(s) = G_1(s) \pm G_2(s) \pm \cdots \pm G_n(s)$$

Blocks in parallel: add (or subtract) their transfer functions.

Negative Feedback Loop

Closed-Loop Transfer Function $$T(s) = \frac{G(s)}{1 + G(s)H(s)}$$

For negative feedback. For positive feedback, the denominator becomes $1 - G(s)H(s)$.

Derivation: Closed-Loop Transfer Function ▸

Let $R(s)$ be the reference input, $E(s)$ the error signal, $Y(s)$ the output, with forward path $G(s)$ and feedback path $H(s)$:

$$E(s) = R(s) - H(s)Y(s)$$ $$Y(s) = G(s)E(s)$$

Substitute $E(s)$ into the second equation:

$$Y(s) = G(s)[R(s) - H(s)Y(s)]$$ $$Y(s) + G(s)H(s)Y(s) = G(s)R(s)$$

Factor and solve:

$$Y(s)[1 + G(s)H(s)] = G(s)R(s) \implies \boxed{T(s) = \frac{Y(s)}{R(s)} = \frac{G(s)}{1+G(s)H(s)}}$$
The denominator $1 + G(s)H(s)$ is called the characteristic polynomial. Its roots (the closed-loop poles) determine system stability. Setting it to zero gives the characteristic equation — the single most important equation in classical control.

Worked Example: Multi-Loop Reduction ▸

Given: Forward path $G_1, G_2, G_3$ in series, inner loop feedback $H_1$ around $G_2$, outer loop feedback $H_2$ from output around entire forward path.

Step 1: Reduce the inner loop first:

$$G_{inner}(s) = \frac{G_2}{1 + G_2 H_1}$$

Step 2: Cascade with remaining blocks:

$$G_{fwd}(s) = G_1 \cdot \frac{G_2}{1 + G_2 H_1} \cdot G_3$$

Step 3: Close the outer loop:

$$T(s) = \frac{G_{fwd}}{1 + G_{fwd} H_2} = \frac{G_1 G_2 G_3}{(1 + G_2 H_1) + G_1 G_2 G_3 H_2}$$

◈ 2.3 — Signal Flow Graphs

A signal flow graph (SFG) uses nodes (representing signals) and directed branches (representing gains/transfer functions) to model the same relationships as block diagrams but in a form suitable for Mason's formula.

Terminology

  • Node: represents a signal variable.
  • Branch: directed edge with gain $G(s)$ from one node to another.
  • Source node: node with only outgoing branches (input).
  • Sink node: node with only incoming branches (output).
  • Forward path: a path from source to sink that visits each node at most once.
  • Loop: a closed path that starts and ends at the same node, visiting no node twice.
  • Non-touching loops: loops that share no common nodes.

◈ 2.4 — Mason's Gain Formula

Mason's Gain Formula $$T(s) = \frac{\sum_{k} P_k \Delta_k}{\Delta}$$

where

$$\Delta = 1 - \sum L_i + \sum L_iL_j - \sum L_iL_jL_k + \cdots$$

$P_k$ = gain of $k$-th forward path
$\Delta$ = graph determinant
$\Delta_k$ = cofactor of $P_k$ (determinant with all loops touching path $k$ removed)
$L_i$ = gain of $i$-th individual loop
$L_iL_j$ = product of gains of $i$-th and $j$-th non-touching loops

Worked Example: Mason's Formula Application ▸

Consider the signal flow graph: $R \xrightarrow{1} x_1 \xrightarrow{G_1} x_2 \xrightarrow{G_2} x_3 \xrightarrow{G_3} Y$ with loops $L_1 = -G_2 H_1$ (at $x_2$-$x_3$) and $L_2 = -G_1 G_2 G_3 H_2$ (outer).

There is one forward path: $P_1 = G_1 G_2 G_3$.

Check non-touching loops: $L_1$ and $L_2$ share node $x_2$, so they touch. No non-touching pairs.

Graph determinant:

$$\Delta = 1 - (L_1 + L_2) = 1 + G_2 H_1 + G_1 G_2 G_3 H_2$$

$\Delta_1$: remove loops touching path 1 — both loops touch it, so $\Delta_1 = 1$.

$$T(s) = \frac{P_1 \Delta_1}{\Delta} = \frac{G_1 G_2 G_3}{1 + G_2 H_1 + G_1 G_2 G_3 H_2}$$
