Block Diagram Reduction & Signal Flow Graphs

Algebra of Interconnected Systems

1. Block Diagram Algebra

Complex control systems consist of many interconnected subsystems. Block diagram algebra provides a systematic way to reduce these into a single equivalent transfer function.

Feedback Loop

For a standard negative feedback loop with forward path $G(s)$ and feedback path $H(s)$, the signals are:

$$ E(s) = R(s) - B(s) $$ $$ B(s) = H(s)Y(s) $$ $$ Y(s) = G(s)E(s) $$

Substitute to solve for $Y(s)/R(s)$:

$$ Y(s) = G(s)[R(s) - H(s)Y(s)] $$ $$ Y(s) + G(s)H(s)Y(s) = G(s)R(s) $$
$$ T(s) = \\frac{Y(s)}{R(s)} = \\frac{G(s)}{1 + G(s)H(s)} $$ (2.1)

2. Signal Flow Graphs (SFG)

Signal flow graphs represent a system as a directed network. Nodes represent variables, and branches (with directions) represent transfer functions (transmittances).

Mason's Gain Formula

A powerful topological method to directly compute the transfer function from any input to any output without sequential reduction.

$$ T = \\frac{1}{\\Delta} \\sum_k P_k \\Delta_k $$ (2.2)
  • $P_k$: the gain of the $k$-th forward path
  • $\\Delta = 1 - \\Sigma L_i + \\Sigma L_i L_j - \\Sigma L_i L_j L_m + \\dots$
  • $\\Delta_k$: the determinant $\\Delta$ evaluated with loops touching path $P_k$ removed.
