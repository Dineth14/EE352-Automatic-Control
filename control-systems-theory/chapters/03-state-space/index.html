State Space Representation

Modern Time-Domain Control Theory

1. The State Equations

Instead of mapping inputs to outputs via a single high-order differential equation, State Space represents a system using $n$ coupled first-order ODEs. The state vector $\mathbf{x}(t)$ encapsulates the complete internal "memory" of the system.

$$ \dot{\mathbf{x}}(t) = A\mathbf{x}(t) + B\mathbf{u}(t) \quad \text{(State Equation)} $$ $$ \mathbf{y}(t) = C\mathbf{x}(t) + D\mathbf{u}(t) \quad \text{(Output Equation)} $$ (3.1)
  • $\mathbf{x} \in \mathbb{R}^n$: State vector
  • $\mathbf{u} \in \mathbb{R}^m$: Input vector
  • $\mathbf{y} \in \mathbb{R}^p$: Output vector
  • $A$: System matrix (encodes eigenvalues/poles)
  • $B$: Input matrix
  • $C$: Output matrix
  • $D$: Feedthrough matrix

2. Controllability and Observability

Can we drive the system to any state using the input $\mathbf{u}$? (Controllability). Can we reconstruct the full state vector $\mathbf{x}$ by only measuring the output $\mathbf{y}$? (Observability).

Controllability Matrix:

$$ \mathcal{C} = \begin{bmatrix} B & AB & A^2B & \dots & A^{n-1}B \end{bmatrix} $$

System is controllable if $\text{rank}(\mathcal{C}) = n$.

Observability Matrix:

$$ \mathcal{O} = \begin{bmatrix} C \\ CA \\ CA^2 \\ \vdots \\ CA^{n-1} \end{bmatrix} $$

System is observable if $\text{rank}(\mathcal{O}) = n$.
