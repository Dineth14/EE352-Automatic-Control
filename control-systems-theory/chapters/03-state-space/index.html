Chapter 03

State Space Analysis

The modern framework for multi-variable dynamic systems

Chapter Overview

State space representation provides a unified mathematical framework for modeling, analyzing, and designing control systems, particularly for multi-input multi-output (MIMO) systems that classical transfer function methods cannot easily handle.

Learning Objectives

  • Formulate state equations from physical systems and transfer functions
  • Convert between state-space and transfer function representations
  • Compute the state transition matrix $e^{At}$
  • Evaluate controllability and observability using rank conditions
  • Transform between canonical forms

◈ 3.1 — State Space Representation

A continuous-time LTI system in state space form consists of two matrix equations describing how the system evolves over time:

State Equations $$\dot{\mathbf{x}}(t) = \mathbf{A}\mathbf{x}(t) + \mathbf{B}\mathbf{u}(t)$$ $$\mathbf{y}(t) = \mathbf{C}\mathbf{x}(t) + \mathbf{D}\mathbf{u}(t)$$

$\mathbf{x} \in \mathbb{R}^n$ — state vector ($n$ states)
$\mathbf{u} \in \mathbb{R}^m$ — input vector ($m$ inputs)
$\mathbf{y} \in \mathbb{R}^p$ — output vector ($p$ outputs)
$\mathbf{A} \in \mathbb{R}^{n\times n}$ — system/state matrix
$\mathbf{B} \in \mathbb{R}^{n\times m}$ — input matrix
$\mathbf{C} \in \mathbb{R}^{p\times n}$ — output matrix
$\mathbf{D} \in \mathbb{R}^{p\times m}$ — feedthrough matrix

Example: RLC Circuit to State Space ▸

For a series RLC circuit with input voltage $u(t)$ and output voltage across capacitor $y = v_C$:

$$L\frac{di}{dt} + Ri + v_C = u(t), \quad i = C\frac{dv_C}{dt}$$

Choose states $x_1 = v_C$ (capacitor voltage) and $x_2 = i$ (inductor current):

$$\dot{x}_1 = \frac{1}{C}x_2, \quad \dot{x}_2 = -\frac{1}{L}x_1 - \frac{R}{L}x_2 + \frac{1}{L}u$$

In matrix form:

$$\begin{bmatrix}\dot{x}_1\\\dot{x}_2\end{bmatrix} = \begin{bmatrix}0 & 1/C\\-1/L & -R/L\end{bmatrix}\begin{bmatrix}x_1\\x_2\end{bmatrix} + \begin{bmatrix}0\\1/L\end{bmatrix}u, \quad y = \begin{bmatrix}1 & 0\end{bmatrix}\begin{bmatrix}x_1\\x_2\end{bmatrix}$$

◈ 3.2 — Transfer Function Conversion

State Space → Transfer Function $$G(s) = \mathbf{C}(s\mathbf{I} - \mathbf{A})^{-1}\mathbf{B} + \mathbf{D}$$
Derivation ▸

Take the Laplace transform of the state equation (zero initial conditions):

$$s\mathbf{X}(s) = \mathbf{A}\mathbf{X}(s) + \mathbf{B}\mathbf{U}(s)$$

Solve for $\mathbf{X}(s)$:

$$(s\mathbf{I} - \mathbf{A})\mathbf{X}(s) = \mathbf{B}\mathbf{U}(s) \implies \mathbf{X}(s) = (s\mathbf{I}-\mathbf{A})^{-1}\mathbf{B}\mathbf{U}(s)$$

Substitute into the output equation:

$$\mathbf{Y}(s) = [\mathbf{C}(s\mathbf{I}-\mathbf{A})^{-1}\mathbf{B} + \mathbf{D}]\mathbf{U}(s)$$

Characteristic Equation

Characteristic Polynomial $$\det(s\mathbf{I} - \mathbf{A}) = 0$$

The eigenvalues of $\mathbf{A}$ are the system poles. Stability requires all eigenvalues to have negative real parts.

◈ 3.3 — State Transition Matrix

Solution of the State Equation $$\mathbf{x}(t) = e^{\mathbf{A}t}\mathbf{x}(0) + \int_0^t e^{\mathbf{A}(t-\tau)}\mathbf{B}\mathbf{u}(\tau)\,d\tau$$

where $\Phi(t) = e^{\mathbf{A}t}$ is the state transition matrix, computed as:

$$e^{\mathbf{A}t} = \mathcal{L}^{-1}\{(s\mathbf{I}-\mathbf{A})^{-1}\} = \sum_{k=0}^{\infty} \frac{(\mathbf{A}t)^k}{k!}$$

◈ 3.4 — Controllability

Controllability Matrix $$\mathcal{C} = \begin{bmatrix}\mathbf{B} & \mathbf{AB} & \mathbf{A}^2\mathbf{B} & \cdots & \mathbf{A}^{n-1}\mathbf{B}\end{bmatrix}$$

The system is controllable if and only if $\text{rank}(\mathcal{C}) = n$.

Physical meaning: A system is controllable if, from any initial state, there exists an input $\mathbf{u}(t)$ that can drive the state to any desired final state in finite time. If the system is not controllable, some internal modes cannot be influenced by the input.

◈ 3.5 — Observability

Observability Matrix $$\mathcal{O} = \begin{bmatrix}\mathbf{C}\\\mathbf{CA}\\\mathbf{CA}^2\\\vdots\\\mathbf{CA}^{n-1}\end{bmatrix}$$

The system is observable if and only if $\text{rank}(\mathcal{O}) = n$.

Physical meaning: A system is observable if the initial state $\mathbf{x}(0)$ can be uniquely determined from the output $\mathbf{y}(t)$ and input $\mathbf{u}(t)$ over a finite time interval. Unobservable modes are hidden from the output.

Duality

The pair $(\mathbf{A},\mathbf{B})$ is controllable if and only if $(\mathbf{A}^T,\mathbf{B}^T)$ is observable, and vice versa with $(\mathbf{A},\mathbf{C})$. This is the controllability-observability duality.

◈ 3.6 — Canonical Forms

Controllable Canonical Form (CCF)

For the transfer function $G(s) = \frac{b_1 s^{n-1}+\cdots+b_n}{s^n+a_1 s^{n-1}+\cdots+a_n}$:

CCF Matrices $$\mathbf{A} = \begin{bmatrix}0&1&0&\cdots&0\\0&0&1&\cdots&0\\\vdots&&&\ddots&\vdots\\0&0&0&\cdots&1\\-a_n&-a_{n-1}&-a_{n-2}&\cdots&-a_1\end{bmatrix}, \;\; \mathbf{B} = \begin{bmatrix}0\\0\\\vdots\\0\\1\end{bmatrix}$$

Observable Canonical Form (OCF)

OCF Matrices $$\mathbf{A} = \begin{bmatrix}0&0&\cdots&0&-a_n\\1&0&\cdots&0&-a_{n-1}\\0&1&\cdots&0&-a_{n-2}\\\vdots&&\ddots&&\vdots\\0&0&\cdots&1&-a_1\end{bmatrix}, \;\; \mathbf{C} = \begin{bmatrix}0&0&\cdots&0&1\end{bmatrix}$$

Diagonal (Modal) Form

If $\mathbf{A}$ has distinct eigenvalues $\lambda_1,\ldots,\lambda_n$, it can be diagonalized via similarity transformation $\mathbf{T}^{-1}\mathbf{A}\mathbf{T} = \text{diag}(\lambda_1,\ldots,\lambda_n)$, where $\mathbf{T}$ is the matrix of eigenvectors.
