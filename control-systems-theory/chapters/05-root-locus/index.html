The Root Locus Method

Visualizing Pole Trajectories

1. Angle and Magnitude Criteria

The Root Locus is the set of all possible roots (closed-loop poles) of the characteristic equation as a system parameter (usually the forward gain $K$) varies from $0 \to \infty$.

For a standard negative-feedback system with open-loop transfer function $G(s)H(s)$, the characteristic equation is:

$$ 1 + K G(s)H(s) = 0 \quad \implies \quad G(s)H(s) = -\frac{1}{K} $$

This single complex equation splits into two criteria for any point $s$ to be on the root locus:

Angle Criterion:

$$ \angle G(s)H(s) = (2q + 1)180^\circ \quad \text{for } q = 0, \pm1, \pm2, \dots $$ (5.1)

Magnitude Criterion:

$$ K = \frac{1}{|G(s)H(s)|} $$ (5.2)

2. Construction Rules

Sketching the root locus by hand relies on several geometric rules, derived directly from the angle criterion:

  • Rule 1: Branches start ($K=0$) at open-loop poles and end ($K=\infty$) at open-loop zeros (or infinity).
  • Rule 2: Real-axis segments exist to the left of an odd number of real-axis poles and zeros.
  • Rule 3: Asymptotes for branches going to infinity intersect at $\sigma_a$ with angles $\theta_a$:
$$ \sigma_a = \frac{\sum P_i - \sum Z_i}{n - m} $$ $$ \theta_a = \frac{(2q + 1)180^\circ}{n - m} $$
