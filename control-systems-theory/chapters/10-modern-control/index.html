Chapter 10

Modern Control

State feedback, observer design, pole placement & LQR

Chapter Overview

Modern control moves beyond classical single-loop design to exploit the full state-space model. This chapter covers state feedback, pole placement, full-order observers, the separation principle, and optimal LQR control.

◈ 10.1 — State Feedback Control

State Feedback Law $$u = -\mathbf{K}\mathbf{x} + r$$

where $\mathbf{K}$ is the $1\times n$ gain vector. The closed-loop dynamics become:

$$\dot{\mathbf{x}} = (\mathbf{A} - \mathbf{B}\mathbf{K})\mathbf{x} + \mathbf{B}r$$

By choosing $\mathbf{K}$, we can place the eigenvalues of $(\mathbf{A}-\mathbf{BK})$ at arbitrary locations, provided the system is controllable.

◈ 10.2 — Pole Placement (Ackermann's Formula)

Ackermann's Formula $$\mathbf{K} = \begin{bmatrix}0 & 0 & \cdots & 1\end{bmatrix} \mathcal{C}^{-1}\, \phi(\mathbf{A})$$

where $\mathcal{C} = [\mathbf{B}\ \ \mathbf{AB}\ \ \cdots\ \ \mathbf{A}^{n-1}\mathbf{B}]$ is the controllability matrix and $\phi(s) = \prod(s - p_i^*)$ is the desired characteristic polynomial evaluated at $\mathbf{A}$.

Why Controllability is Required ▸

If rank$(\mathcal{C}) < n$, then some state directions are unreachable from the input. These modes cannot be moved by any feedback gain. The system is only "stabilizable" if the uncontrollable modes are already stable.

◈ 10.3 — State Observers

In practice, not all states are measured. A full-order observer (Luenberger observer) estimates the state:

Luenberger Observer $$\dot{\hat{\mathbf{x}}} = \mathbf{A}\hat{\mathbf{x}} + \mathbf{B}u + \mathbf{L}(y - \mathbf{C}\hat{\mathbf{x}})$$

The estimation error $\mathbf{e}=\mathbf{x}-\hat{\mathbf{x}}$ satisfies $\dot{\mathbf{e}} = (\mathbf{A}-\mathbf{LC})\mathbf{e}$.

By choosing $\mathbf{L}$, we place the eigenvalues of $(\mathbf{A}-\mathbf{LC})$ — requiring observability.

Separation Principle

The controller gain $\mathbf{K}$ and observer gain $\mathbf{L}$ can be designed independently. The combined (observer + state feedback) system has eigenvalues equal to the union of eig$(A-BK)$ and eig$(A-LC)$.

◈ 10.4 — Linear Quadratic Regulator (LQR)

LQR Cost Function $$J = \int_0^\infty \!\big(\mathbf{x}^T\mathbf{Q}\mathbf{x} + u^T\mathbf{R}\,u\big)\,dt$$

The optimal gain $\mathbf{K}^* = \mathbf{R}^{-1}\mathbf{B}^T\mathbf{P}$ where $\mathbf{P}$ solves the algebraic Riccati equation:

$$\mathbf{A}^T\mathbf{P} + \mathbf{P}\mathbf{A} - \mathbf{P}\mathbf{B}\mathbf{R}^{-1}\mathbf{B}^T\mathbf{P} + \mathbf{Q} = \mathbf{0}$$

LQR provides an automatic, optimal gain that guarantees infinite gain margin and $\geq$ 60° phase margin when $\mathbf{R}$ is scalar.
