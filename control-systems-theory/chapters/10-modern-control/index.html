Modern Control

Optimal Control & State Estimation

1. Linear Quadratic Regulator (LQR)

Unlike PID control, which is often tuned heuristically, optimal control seeks to minimize a cost function. The Linear Quadratic Regulator (LQR) computes an optimal state-feedback gain matrix $K$ that minimizes an infinite-horizon quadratic cost function $J$:

$$ J = \int_0^\infty \left( x^T Q x + u^T R u \right) dt $$ (10.1)

Where:

  • $Q$: State cost weight matrix (penalizes deviation from desired state).
  • $R$: Control cost weight matrix (penalizes actuator effort).

Interactive: LQR Weight Tuning

Adjust the relative weights of $Q$ (State Penalty) and $R$ (Effort Penalty). Notice the trade-off: a high state penalty forces a faster response but requires enormous control effort, while a high effort penalty saves energy but slows the response.

2. The Kalman Filter

In the real world, sensors are noisy, and we cannot directly measure all state variables. The Kalman Filter is an optimal estimator that fuses a mathematical model of the system with noisy sensor data to produce the best possible estimate of the true state $\hat{x}$.

Predict Step:

$$ \hat{x}_{k|k-1} = A \hat{x}_{k-1|k-1} + B u_k $$

Update Step:

$$ \hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k(y_k - C \hat{x}_{k|k-1}) $$ (10.2)
